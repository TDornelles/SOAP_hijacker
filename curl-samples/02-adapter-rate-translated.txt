# Adapter: translated Rate operation (SOAP in -> GLP -> SOAP out)
#
# Exercises the full hijack path: the adapter parses rData, POSTs the package to GLP's
# public rates endpoint, and rebuilds the legacy-shaped SOAP response.
# Run from the repo root (the clone) so the fixture path resolves.
# WSKEY in the fixture is a dummy - the ADAPTER does not validate it (GLP Rate is public),
# so the dummy is fine here, unlike against the real legacy service (see 04).

curl.exe -s -X POST http://localhost:8081/Routing/Service/Soap/V2.6/Routing.svc -H "Content-Type: text/xml; charset=utf-8" -H "SOAPAction: \"http://tempuri.org/IRouting/RateLandedCost\"" --data-binary "@fixtures/rate-request.xml"

#     -H 'SOAPAction: "http://tempuri.org/IRouting/RateLandedCost"' \

#       <a:MessageEntities>...<a:Code>00000</a:Code><a:Description>Request Successful</a:Description>...

# NOTE: FreightCost is GLP's fuel-inclusive TotalFreightCost by design (decided 2026-07-15),
